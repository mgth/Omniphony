# Omniphony

Monorepo de packaging et de distribution pour la suite Omniphony.

Contenu actuel :

- `gsrd/`
- `SpatialVisualizer/`
- `docs/`
- `linux/`
- `windows/`
- `assets/`
- `scripts/`

Objectif :

- préparer une distribution unifiée Linux et Windows
- regrouper les composants nécessaires à l'installation
- faire converger progressivement les anciens dépôts vers ce dépôt unique
